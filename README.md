# AWS Terraform Web Server

This project demonstrates how to deploy a web server on AWS using Terraform.

## Technologies

- AWS
- Terraform
- EC2

## Infrastructure

Terraform creates:

- EC2 instance
- Cloud infrastructure using Infrastructure as Code

## Terraform Commands

Initialize Terraform

terraform init

Plan infrastructure

terraform plan

Apply infrastructure

terraform apply
